# PrepTrack

PrepTrack is a Java desktop application that helps students track their DSA and coding practice.

## Features

- Login Screen
- Dashboard
- Add Problems
- View Problems
- Java Swing GUI
- Object-Oriented Programming
- File Handling

## Technologies Used

- Java
- Java Swing
- OOP
- Collections Framework
- File Handling
- VS Code

## Project Structure

```
src/
 ├── model/
 ├── service/
 ├── storage/
 ├── ui/
 └── Main.java
```

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Run `Main.java`.
